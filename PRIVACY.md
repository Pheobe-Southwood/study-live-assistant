# 隐私说明

自习直播助手是纯本地单机程序，不要求注册或登录，也不包含遥测、广告、分析 SDK、云同步或自动上传功能。

任务内容、分类、学习时长、倒数日、外观设置和快捷键均保存在 `%LocalAppData%\StudyLiveAssistant`。用户选择的背景图片会复制到该目录的 `Assets` 文件夹。

程序发生错误时，会在本地 `Logs` 文件夹写入异常信息。日志不会自动发送；如果用户主动在 GitHub Issue 中提供日志，应先检查其中是否包含不希望公开的本机路径或任务内容。

直播窗口中的文字会被 OBS 等采集软件捕获，这是程序的预期用途。配置窗口与学习控制窗口未启用 Windows 屏幕捕获排除，请在直播软件中使用窗口采集并自行检查场景内容。
