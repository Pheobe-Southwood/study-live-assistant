# 使用说明

## 建立当天任务

打开“学习任务管理”，选择日期、一级类型、二级类型、目标开始时间和任务目标。目标开始时间只用于计划与排序；第一次点击播放时，程序会自动记录不可手填的实际开始时间。

时间型任务使用“分钟”作为目标，播放后自动推进；计数型任务可选择章节、页数、题目或自定义单位，通过加减按钮推进。切换进度类型后，单位下拉框只显示与该类型兼容的选项。

每项任务都会记录实际用时。计数型任务的实际用时不会代替页数或题目进度。

任务按目标开始时间排序；相同时间按创建顺序排列。可通过“复制前一天任务”快速建立新一天的计划，复制后的任务从未开始状态重新计数，也不会复制实际开始时间。

## 直播窗口与 OBS

1. 在“基础配置”选择 1920×1080、1280×720 或输入自定义尺寸。
2. 设置绿幕色、信息区背景、字体与顶部信息项目，点击保存。
3. 回到首页点击“启动窗口”。
4. 在 OBS 中添加“窗口采集”，选择标题为“自习直播主页面”的窗口。
5. 根据直播布局对绿幕区域使用色度键滤镜。

直播窗口不可缩放。在信息区或绿幕区按住鼠标左键即可拖动，位置会被保存；也可在基础配置中直接填写窗口 X/Y 坐标。

“关闭窗口”会隐藏直播窗口而不是销毁它。只要“自习直播助手”仍在运行，再次点击“启动窗口”就会复用原窗口句柄，OBS 中已经建立的窗口采集通常会自动恢复。完全退出程序后 Windows 会重新分配窗口句柄，此时 OBS 将通过稳定的程序名和窗口标题重新匹配。

## 学习模式

点击“开启学习模式”后，配置窗口隐藏，出现置顶的紧凑控制窗口。窗口显示当前任务、实际用时、目标进度和下一任务类型。

- `‹`：上一任务；
- `−`：时间型减 1 分钟，计数型减一个配置步长；
- `▶ / Ⅱ`：开始或暂停；
- `+`：时间型加 1 分钟，计数型加一个配置步长；
- `›`：下一任务；
- `✓`：将当前任务标记为完成；
- “配置”：返回配置窗口。

切换任务会暂停计时，新任务不会自动开始。达到目标只做状态提示，不自动跳转。

## 默认全局快捷键

| 操作 | 快捷键 |
| --- | --- |
| 播放 / 暂停 | `Ctrl+Alt+Space` |
| 上一任务 | `Ctrl+Alt+Left` |
| 下一任务 | `Ctrl+Alt+Right` |
| 进度加一 | `Ctrl+Alt+Up` |
| 进度减一 | `Ctrl+Alt+Down` |
| 显示 / 隐藏学习窗口 | `Ctrl+Alt+S` |

可以在“快捷键”页面重新绑定。若组合重复或已被其他程序占用，程序会拒绝新配置并恢复原绑定。

## 备份与清理

当前版本不提供云同步和内置导入导出。退出程序后，可手动复制 `%LocalAppData%\StudyLiveAssistant` 完成完整备份。

若要清除全部数据，请先退出程序，再删除上述目录。此操作无法撤销。
